5. Сетевые модули не привязанные к конкретной ОС
================================================

Для работы через CLI, Ansible поддерживает такие модули, не привязанные к конкретной ОС: 

* cli_command - выполняет команды show 
* cli_config - выполняет команды конфигурации

.. note::
    
    Модули cli_command, cli_config появились в Ansible 2.7.
